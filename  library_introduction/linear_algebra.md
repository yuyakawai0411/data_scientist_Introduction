## 逆行列の意味

- ある数 \* 逆数 = 1
- 係数行列 \* 逆行列 = 単位行列

連立方程式の解を解きやすくすることができる。
係数行列 A に逆行列 invA をかけて単位行列 E にすれば、x=invA\*B になり x の値が簡単にもとまる。

```
# 連立方程式
x - y = 2
3x + 4y = 8
# 連立方程式を行列で表す
(1 1) (x) = (2)
(3 4) (y)   (8)
----- ---   ---
  A    x     B
```
